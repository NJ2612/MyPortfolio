# Naina Joshi Portfolio

A high-performance, system-themed portfolio built with React and Vite.

## Tech Stack
- **Framework**: React (Vite)
- **Styling**: Vanilla CSS (Variables, Glassmorphism, HUD System)
- **Animation**: Framer Motion
- **Icons**: Lucide React

## Project Structure
```
/src
 ├─ components/     # Modular UI components (Loader, Hero, Projects, etc.)
 ├─ data/           # Static content (projects.js, skills.js)
 ├─ pages/          # Main page views
 └─ App.jsx         # Entry point with Loader logic
```

## Getting Started

1. **Install Dependencies**
   ```bash
   npm install
   ```

2. **Run Development Server**
   ```bash
   npm run dev
   ```

3. **Build for Production**
   ```bash
   npm run build
   ```

## Customization
- Update `src/data/projects.js` to add more case studies.
- Adjust colors in `src/index.css` via CSS variables.
